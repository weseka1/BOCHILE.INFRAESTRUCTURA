# Histórico de Bootstrap del Excel Maestro

Versiones anteriores del Apps Script que arma la estructura inicial del Sheet maestro. **La versión vigente es `../Bootstrap_FINAL.gs`** en la carpeta padre.

Si necesitás recrear el Sheet desde cero (por ejemplo en otra cuenta Google), usá `Bootstrap_FINAL.gs`, NO estos.

| Archivo | Estado |
|---|---|
| `Bootstrap_Apps_Script.gs` | v1, deprecated |
| `Bootstrap_Apps_Script_v2.gs` | v2, deprecated |
| `Bootstrap_Apps_Script_v3.gs` | v3, deprecated |
| (carpeta padre) `Bootstrap_FINAL.gs` | **vigente** |

Quedan acá como auditoría de la evolución del diseño.
