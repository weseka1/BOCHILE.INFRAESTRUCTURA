# Bochile · Resumen Ejecutivo del Proyecto

## Lo que estamos construyendo

Una infraestructura de ventas con IA end-to-end para Inmobiliaria Bochile, primera implementación de este tipo en Bahía Blanca. La propuesta combina cinco piezas que normalmente se venden por separado:

1. **Web inmobiliaria** con tours 360° y catálogo dinámico
2. **CRM** con pipeline kanban y scoring automático de leads
3. **Dashboard ejecutivo** con KPIs en vivo
4. **IA Vendedora** (Camila) conectada a WhatsApp, web y ZonaProp
5. **Administración automatizada de alquileres** (cobranza, reclamos, ajustes IPC)

## La promesa al cliente

**80% de tiempo humano ahorrado.** Bochile mantiene el equipo actual pero deja de hacer trabajo repetitivo. Las horas se redirigen a cierres reales.

## Por qué esto rompe parámetros en BB

- Ninguna inmobiliaria de Bahía Blanca tiene tours 360° + IA conversacional + automatización de alquileres en una sola plataforma
- Bahía Blanca está en pleno auge inmobiliario (zona Patagonia, Universitario, Palihue)
- ZonaProp ya está saturado — quien tiene mejor experiencia digital se queda con el lead
