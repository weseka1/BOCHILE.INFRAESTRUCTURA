# Assets · Bochile

Carpeta para todo lo que no es documento markdown:

- `logos/` — Logo Bochile en distintos formatos (PNG, SVG, blanco/negro)
- `propiedades/` — Fotos y tours 360° por propiedad
- `vendedores/` — Fotos del equipo (con permiso)
- `creatividades/` — Reels, carousels, stories producidos
- `referencias/` — Inspiración, competencia, mood boards
- `legal/` — Templates de consentimiento, contratos firmados
- `_archivo/` — Material obsoleto o reemplazado (con fecha)

## Convenciones de nombres

```
propiedades/{barrio}_{calle-numero}/
   ├── tour_360.zip
   ├── plano.pdf
   ├── fotos/
   │   ├── 01_living.jpg
   │   ├── 02_cocina.jpg
   │   └── ...
   └── ficha.md
```

```
creatividades/{YYYY-MM-DD}_{ad-num}_{pilar}_{formato}.{ext}
   ej: 2026-05-08_ad-01_tour-360_reel-30s.mp4
```

```
legal/{YYYY-MM-DD}_{tipo}_{persona}.pdf
   ej: 2026-05-15_consentimiento_familia-beltran.pdf
```
